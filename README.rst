安装
********

目前 lightweb 支持的 Python 环境有 3.4+


为了简化安装过程，推荐使用 pip 进行安装

.. code-block:: bash

    pip install lightweb

升级 lightmail 到新版本::

    pip install -U lightweb

如果需要安装 GitHub 上最新代码::

    pip install https://github.com/ni-ning/lightweb/archive/master.zip
